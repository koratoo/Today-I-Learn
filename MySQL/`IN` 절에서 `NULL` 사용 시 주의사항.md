### `IN` 절에서 `NULL` 사용 시 주의사항

SQL에서 `IN` 절을 사용할 때 `NULL` 값을 포함하는 경우, 의도한 대로 동작하지 않을 수 있다.

#### ❌ 잘못된 예시:
```sql
SELECT * FROM 테이블명 WHERE 컬럼명 IN ('컬럼값', NULL);
```
- 위의 쿼리는 사실상 아래와 동일하게 동작한다.
```sql
SELECT * FROM 테이블명 WHERE 컬럼명 IN ('컬럼값');
```
- 즉, `NULL` 값이 포함되어 있어도 `NULL` 자체는 비교 대상에서 제외된다.
- 따라서 `컬럼명`이 `NULL`인 데이터는 조회되지 않는다.

#### ✅ 올바른 방식:
`NULL`을 포함하여 데이터를 조회하려면 `IS NULL` 조건을 별도로 추가해야 한다.
```sql
SELECT * FROM 테이블명 WHERE 컬럼명 IN ('컬럼값') OR 컬럼명 IS NULL;
```
이렇게 하면 `컬럼명`이 `'컬럼값'`이거나 `NULL`인 경우 모두 조회할 수 있다.

#### 🔍 정리
- `IN` 절에서는 `NULL`이 포함되어도 무시된다.
- `NULL` 값을 고려한 검색을 하려면 `IS NULL`을 별도로 명시해야 한다.
